Section 7 - Arrays

Arrays

Arrays are data structures that have a fixed number of elements contained within. 

  int my_array[4] = {2,3,1,4};

We must know the length of the array in advance as well as the type. We are also flexible to use multiple operations on elements of an array, for example consider, 

  int a[4] = {3,4,5,1};

  int j = a[0];
  int *p = &a[3];

  a[2] = a[a[3]];
  a[2]++;

Initialization

You can initialize an array completely or partially (or empty) as follows 

     int a[3] = {};
     int b[2] = {1,2};
     int v[5] = {1,2,3}; // REST ARE ZERO AUTOMATICALLY

Uninitialized global arrays (LIKE INTEGERS) will be initialized to zero, whereas (LIKE INTEGERS) uninitialized local arrays will just have random values from the previous stack.

Array Size

  • The length of an array is the number of elements in the array.
  • The size of an array is the number of bytes it occupies.
int a[6] = {};

The array above will occupy, (6 x sizeof(int)) = 24 bytes.

The Array Identifier

An array identifier by itself does not actually have any intrinsic purpose, instead it just simply referees to the memory address of the array, which also happens to be the address of the first element. 

  int a[6] = {4,5,6,7,6,5};

  trace_ptr(a);
  trace_ptr(&a);
  trace_ptr(&a[0]);

  // trace_ptr(a) = trace_ptr(&a) = trace_ptr(&a[0])

Given that the using the identifier of the array on its own is equivalent to the the address of the first element, we can leverage this to extract (dereference) the value of the first element, 

  int a[3] = {0,2,3};

  trace_int(*a);
  trace_int(a[0]);

  // trace_int(*a) =  trace_int(a[0]);

Function Syntax with Arrays

Use the following syntax for declaring an array as an argument (const or not) 

// my_sum(arr, len) Computes sum of contents of the array
  int my_sum(int arr[], int len);

Note that since we can only pass the address of an array, we have to pass a separate variable to keep track of the length of the array.

Pointer Arithmetic

  • Given a pointer p, the expression (p + 1) will point to the next item in memory. The specific expression is given as,

\((p + i) = p + i \cdot \text{sizeof}(*p)\). Subtraction and decremantation/incrimination works the same way.

  • You cannot add two pointers
  • You can subtract two pointers of the same type and the expression will be evaluated to, \(p - q = \frac{p-q}{\text{sizeof}(*p)}\)
  • You can also do standard comparison operations; <, <=, !, =, >, >

Pointer Arithmetic and Arrays

Pointer arithmetic is precisely the manner in which elements are extracted from an array, for example. 

  int a[3] = {5,3,2};

  int j = a[1];
  int i = *(a + 1);

  assert(i == j);
  // works

  int k = &a[2];
  int m = (a + 2);

  assert(k == m);
  // works

Example, summing array with pointer arithmetic

We can implement the sumarray algorithm using strictly pointer syntax. 

     int sum_array(const int *a, int len){
       int sum = 0;
       for(int *p = a; p < a + len; p++){
     sum += *p;
       }

       return sum;
     }

Selection Sort

The following is the Selection Sort Algorithm, 

  void selection_sort(int a[], int len){
    int pos = 0;
    for(int i = 0 ; i < len; i++){
      pos = i;
      for(int j = i + 1; j < len; j++){
    if(a[j] < pos){
      pos = j;
    }
      }
      swap(&a[i], &a[pos]);
    }
  }

Insertion Sort

The following is the Insertion Sort Algorithm, 

  void insertion_sort(int a[], int len){
    for(int i = 1; i < len; i++){
      for(int j = i; j > 0 && a[j - 1]> a[j]; j--){
    swap(&a[j], &a[j - 1]);
      }
    }
  }

Linear Search

Standard linear searching algorithm for finding some element (first one if duplicates exist) in an array 

  int lin_search(int target, const int a[], int len){
    for(int x = 0 ; x < len; x++){
      if(a[x] == target){
    return x;
      }
    }
    return -1;
  }

Binary Search

Standard binary search algorithm for finding some element (first one if duplicates exist) in an array USE IFF ARRAY IS SORTED 

  int binary_search(int target, const int a[], int len){
    int HIGH = 0;
    int LOW = len - 1;

    while(LOW <= HIGH){
      int mid = (LOW + HIGH) / 2;

      if(a[mid] == target){
    return mid;
      } else if(a[mid] < target){
    LOW = mid + 1;
      } else {
    HIGH = mid - 1;
      }
    }

    return -1;
  }

Multidimensional Array

We cannot explicitly define a 2D array (like Racket) but we can play around with syntax to get the job done. Suppose for a \(n \times m\), array \(A\), I wanted to access the element \(A_{ij}\), aka the element in row \(i\) and col \(j\), where \(0 \leq i < n\), \(0 \leq j < m\), then the formula is A[i * m + j]

Oversized Arrays

Because in C we must know the length of the array in advance when we declare it, a common strategy is to simply declare the array with an OVERSIZED length, which can allow for flexibility in the future to add elements to the list should we choose to do so. (Try to avoid because this can really waste memory).
